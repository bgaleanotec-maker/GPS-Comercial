Vision General del Sistema

GPS Comercial es una plataforma integral de seguimiento de flotas y gestion comercial desarrollada para Vanti, empresa colombiana del sector de gas natural. El sistema permite monitorear equipos de campo en tiempo real, automatizar la deteccion de visitas comerciales, gestionar tareas y generar analitica avanzada para la toma de decisiones.

Capacidades Clave

Rastreo GPS en Tiempo Real

Monitoreo continuo de vehiculos y empleados en campo con registro de rutas, distancias y velocidades.

Deteccion Automatica de Visitas

Geocercas configurables que detectan automaticamente la llegada del empleado al aliado comercial.

Gestion de Tareas y Plantillas

Asignacion, seguimiento y validacion GPS de tareas con generacion recurrente automatizada.

Scoring y Analitica

Puntaje de conduccion, KPIs de negocio, exportacion a Excel y comparativas por periodo.

Modulos del Sistema

Dashboard GPS

Monitoreo de flota en tiempo real con mapa interactivo. Visualiza la ubicacion de todos los empleados, distancia recorrida por jornada, analisis de velocidad promedio y maxima. Incluye historial de rutas y alertas de exceso de velocidad.

Tiempo real Mapa Distancia Velocidad

Torre de Control

Panel centralizado para monitorear el cumplimiento de tareas de todo el equipo. Filtros por prioridad (alta, media, baja), estado (pendiente, en progreso, completada) y empleado. Vista consolidada para lideres con indicadores de cumplimiento diario.

Monitoreo Filtros Cumplimiento

Mi Negocio (Lider)

Dashboard ejecutivo para lideres de negocio. KPIs principales: visitas realizadas vs. meta, ranking de empleados por cumplimiento, metricas de productividad del equipo. Comparativa semanal y mensual con indicadores de tendencia.

KPIs Ranking Metricas

Agenda / Cronograma

Calendario semanal de tareas asignadas con vista por dia. Cada tarea muestra aliado destino, prioridad y estado. Permite completar tareas con validacion GPS automatica (verificar que el empleado esta en la ubicacion) o marcado manual con observaciones.

Calendario Validacion GPS Manual

Plantillas Recurrentes

Sistema de generacion automatica de tareas basado en plantillas. El lider define una plantilla con frecuencia (diaria, semanal, mensual), empleado asignado, aliado destino y prioridad. El sistema genera las tareas automaticamente segun la programacion configurada.

Automatico Diario Semanal Mensual

Gestion de Visitas

Registro integral de visitas comerciales. Deteccion automatica via geocerca cuando el empleado ingresa al radio del aliado, o registro manual con formulario. Permite adjuntar evidencia fotografica y observaciones. Historial completo de visitas por aliado y empleado.

Geocerca Manual Evidencia

Analitica

Modulo de inteligencia de negocio con dashboards interactivos. Permite exportar datos a Excel, comparar periodos (semanal, mensual), visualizar tendencias de cumplimiento y generar reportes detallados por empleado, aliado o equipo completo.

Dashboards Excel Comparativas

Scoring de Conduccion

Sistema de calificacion de conduccion basado en reglas configurables. Detecta infracciones como exceso de velocidad, frenado brusco y aceleracion agresiva. Cada infraccion resta puntos segun severidad. Score de 0 a 100 con historico y tendencias.

Infracciones Penalizacion Score 0-100

Alertas WhatsApp

Notificaciones automaticas via WhatsApp usando la API de Ultramsg. Envia alertas de emergencia (accidentes, paradas prolongadas), reportes diarios consolidados al lider, y notificaciones de eventos criticos en tiempo real.

Emergencias Reportes diarios Ultramsg

Gestion de Usuarios

Administracion completa de usuarios del sistema. Creacion, edicion y desactivacion de cuentas. Asignacion de roles (administrador, lider, empleado), gestion de estado (activo, inactivo, vacaciones) y asociacion con dispositivos GPS de Traccar.

CRUD Estados Dispositivos

Aliados Comerciales

Gestion de puntos comerciales (aliados) de Vanti. Cada aliado tiene coordenadas GPS, direccion, y radio de geocerca configurable para la deteccion automatica de visitas. Permite ajustar el radio segun las condiciones fisicas del punto.

Geocerca Radio Coordenadas

API Keys

Modulo de gestion de tokens para integraciones externas. Permite generar, revocar y monitorear API Keys que habilitan el acceso programatico al sistema. Cada key tiene permisos configurables y registro de uso.

Tokens Integraciones Permisos

Flujos de Trabajo Detallados

A

Registro de Visita Automatica (GPS)

1

El empleado inicia su jornada laboral

La aplicacion OsmAnd en su dispositivo comienza a enviar posiciones GPS al servidor Traccar de forma continua.

2

GPS Comercial consulta posiciones del servidor Traccar

El sistema obtiene las coordenadas actuales del empleado y las compara contra las geocercas de todos los aliados comerciales asignados.

3

La geocerca detecta que el empleado ingreso al radio del aliado

Se calcula la distancia entre la posicion GPS y las coordenadas del aliado. Si es menor al radio configurado (ej. 100m), se activa la deteccion.

4

Se valida el intervalo minimo entre visitas

El sistema verifica que no se haya registrado una visita al mismo aliado dentro del intervalo configurado, evitando duplicados.

5

Visita registrada automaticamente en la base de datos

Se crea el registro con fecha, hora, empleado, aliado, coordenadas y tipo "automatica". Queda disponible en reportes y analitica.

B

Asignacion de Tarea por Lider

1

El lider accede al modulo de gestion de tareas

Desde la Torre de Control o la agenda, selecciona "Nueva tarea" y escoge el empleado, aliado destino, fecha y prioridad.

2

Define detalles de la tarea

Completa titulo, descripcion, observaciones y selecciona si requiere validacion GPS (presencia fisica en el aliado).

3

La tarea aparece en el cronograma del empleado

El empleado ve la tarea en su agenda semanal, con indicacion de fecha limite, aliado asignado y nivel de prioridad.

4

El lider monitorea el cumplimiento desde la Torre de Control

La tarea cambia de estado conforme el empleado la ejecuta: pendiente, en progreso, completada (con o sin validacion GPS).

C

Generacion de Tareas Recurrentes

1

El lider crea una plantilla recurrente

Define nombre, empleado, aliado, prioridad y frecuencia: diaria (L-V), semanal (dia especifico) o mensual (dia del mes).

2

El sistema ejecuta el job de generacion

Un proceso programado revisa las plantillas activas y genera automaticamente las tareas correspondientes al periodo actual.

3

Se verifican duplicados

Antes de crear la tarea, el sistema valida que no exista una tarea identica ya generada para la misma fecha, evitando duplicacion.

4

Las tareas aparecen automaticamente en la agenda del empleado

Sin intervencion manual, las tareas generadas estan listas para ejecucion con todos los datos de la plantilla.

D

Validacion de Tarea por GPS

1

El empleado intenta completar una tarea desde su cronograma

Selecciona la tarea y pulsa "Completar con validacion GPS".

2

El sistema obtiene la posicion GPS actual del empleado

Consulta la ultima posicion reportada por el dispositivo del empleado en Traccar.

3

Compara la posicion contra la geocerca del aliado

Calcula la distancia entre la ubicacion actual y las coordenadas del aliado asignado en la tarea.

4

Resultado: Aprobada o Rechazada

Si esta dentro del radio, la tarea se marca como completada con validacion GPS exitosa. Si no, se notifica al empleado que debe estar en la ubicacion del aliado.

E

Envio de Reporte Diario

1

Se activa el job programado al final de la jornada

Segun la hora configurada en el sistema (ej. 18:00), se ejecuta el proceso de consolidacion diaria.

2

Se recopilan datos del dia para cada empleado

Tareas completadas, visitas realizadas, distancia recorrida, alertas generadas y score de conduccion.

3

Se genera el mensaje de reporte consolidado

Se formatea un resumen con metricas clave del equipo, ordenado por cumplimiento.

4

Se envia el reporte por WhatsApp al lider

Usando la API de Ultramsg, se envia el resumen al numero de WhatsApp del lider de negocio configurado.

F

Alerta de Emergencia

1

El sistema detecta un evento critico en tiempo real

Exceso de velocidad extremo, frenado brusco severo, o inactividad prolongada fuera de horario normal.

2

Se evalua la severidad segun las reglas configuradas

El motor de reglas clasifica el evento y determina si amerita una alerta inmediata o solo un registro.

3

Se genera y envia la alerta por WhatsApp

Mensaje de emergencia con tipo de evento, empleado, ubicacion (enlace a mapa) y hora exacta, enviado al lider y al administrador.

4

Se registra el incidente y se actualiza el scoring

La infraccion queda en el historial del empleado, se descuentan puntos del score de conduccion y el evento se refleja en analitica.

Configuracion del Sistema

Parametros Generales

Parametro Descripcion Ejemplo
start_time Hora de inicio de la jornada laboral 07:00
end_time Hora de fin de la jornada laboral 18:00
active_days Dias habiles de la semana (0=Lunes, 6=Domingo) 0,1,2,3,4
visit_interval Intervalo minimo en minutos entre visitas al mismo aliado 60
report_hour Hora de envio del reporte diario por WhatsApp 18:30
geocerca_radius Radio por defecto de deteccion de visita (metros) 100

Integracion WhatsApp (Ultramsg)

Credenciales requeridas

  • ULTRAMSG_INSTANCE_ID - Identificador de la instancia
  • ULTRAMSG_TOKEN - Token de autenticacion API
  • ADMIN_PHONE - Telefono del administrador

Tipos de mensajes

  • Alertas de emergencia (inmediatas)
  • Reporte diario consolidado (programado)
  • Notificaciones de infracciones (por evento)

Alertas de Emergencia

Las alertas de emergencia se activan cuando se detectan eventos criticos que requieren atencion inmediata. Se configuran mediante reglas que definen umbrales y condiciones de activacion.

Velocidad Extrema

Supera el umbral maximo configurado (ej. 120 km/h)

Frenado Brusco

Desaceleracion que excede el limite seguro

Inactividad

Sin reporte GPS por tiempo prolongado

Integraciones

T

Traccar GPS Server

Servidor de rastreo vehicular

Funcion: Recibe y almacena posiciones GPS de todos los dispositivos en campo.

Protocolo: OsmAnd (compatible con app Android/iOS del mismo nombre).

Comunicacion: GPS Comercial consulta la API REST de Traccar para obtener posiciones, dispositivos y eventos.

Datos: Latitud, longitud, velocidad, altitud, timestamp, atributos del dispositivo.

W

Ultramsg WhatsApp API

Mensajeria instantanea

Funcion: Envio de mensajes WhatsApp automatizados a lideres y administradores.

Endpoint: API REST con autenticacion via Instance ID y Token.

Usos: Alertas de emergencia, reportes diarios consolidados, notificaciones de infracciones.

Formato: Numeros en formato internacional colombiano (57XXXXXXXXXX).

S

SendGrid Email

Correo electronico transaccional

Funcion: Envio de correos electronicos del sistema (restablecimiento de contrasena, notificaciones).

Autenticacion: API Key de SendGrid configurada en variables de entorno.

Plantillas: Emails HTML con formato corporativo de Vanti.

Uso: Complementario a WhatsApp para comunicaciones formales.

P

PostgreSQL Database

Base de datos relacional

Funcion: Almacenamiento persistente de todos los datos del sistema.

ORM: SQLAlchemy con migraciones via Flask-Migrate.

Hosting: Servicio gestionado en la nube (compatible con Render, AWS RDS, etc.).

Tablas principales: users, allies, visits, tasks, templates, scoring_rules, settings.

Glosario de Terminos

Geocerca

Perimetro virtual circular definido por coordenadas GPS y un radio en metros. Se utiliza para detectar automaticamente cuando un empleado entra o sale de la zona de un aliado comercial.

Aliado Comercial

Punto de atencion o establecimiento comercial asociado a Vanti. Cada aliado tiene una ubicacion GPS, direccion y radio de geocerca para la deteccion de visitas.

Visita Automatica

Registro de visita generado automaticamente por el sistema cuando la geocerca detecta que el empleado ingreso al radio del aliado, sin intervencion manual.

Validacion GPS

Proceso de verificacion que confirma la presencia fisica del empleado en la ubicacion del aliado al momento de completar una tarea, usando las coordenadas GPS en tiempo real.

Plantilla Recurrente

Configuracion que define una tarea con frecuencia automatica (diaria, semanal, mensual). El sistema genera las tareas correspondientes sin intervencion del lider.

Score de Conduccion

Calificacion numerica (0 a 100) que refleja la calidad de conduccion del empleado. Se calcula restando puntos por infracciones detectadas (velocidad, frenado, aceleracion).

Torre de Control

Panel centralizado del lider de negocio que permite monitorear en tiempo real el estado de cumplimiento de todas las tareas asignadas a su equipo.

OsmAnd

Aplicacion movil de navegacion y rastreo GPS. Utilizada como cliente en los dispositivos de los empleados para enviar posiciones al servidor Traccar.

Traccar

Plataforma de codigo abierto para rastreo GPS. GPS Comercial lo utiliza como servidor de posiciones, consultando su API REST para obtener ubicaciones en tiempo real.

Ultramsg

Proveedor de API para WhatsApp Business. Permite enviar mensajes programaticos de texto a numeros de WhatsApp desde el backend del sistema.

Jornada Laboral

Periodo de tiempo configurado (start_time a end_time) durante el cual el sistema monitorea activamente a los empleados y registra actividades.

API Key

Token de autenticacion que permite a sistemas externos acceder a los datos de GPS Comercial de forma programatica y segura.
